أنواع الـ Toast:
   🟢 Success (أخضر) - عمليات ناجحة
   🔴 Error (أحمر) - أخطاء
   🟡 Warning (أصفر) - تحذيرات
   🔵 Info (أزرق) - معلومات

───────────────────────────────────────────────────────────────────
4️⃣  التنبيهات عند وصول طلب جديد
───────────────────────────────────────────────────────────────────

عند وصول طلب جديد، سيحدث:

   1. 🎨 Toast Notification
      - داخل التطبيق
      - "New order from Room 101"
      - لا تختفي تلقائياً (حتى يغلقها)

   2. 🔔 Browser Notification
      - خارج التطبيق
      - تظهر حتى لو التاب مخفي
      - "🔔 New Order Received!"
      - "Room: 101, Items: 3"

   3. 🔊 Sound
      - صوت تنبيه قصير
      - يشتغل حتى لو التاب في الخلفية

   4. 📳 Vibration (للأجهزة المدعومة)
      - اهتزاز بسيط

الفائدة:
   ✓ لن يفوت أي طلب!
   ✓ حتى لو التاب مخفي
   ✓ مناسب للتابلت تماماً
   ✓ احترافي

🐳 بخصوص Docker
───────────────────────────────────────────────────────────────────

✅ عادي تماماً تكمل بـ Docker!

الاستخدام:
   - PostgreSQL فقط في Docker
   - Backend مع PM2 (خارج Docker)
   - Frontend static files مع Nginx

الأوامر:
   1. تشغيل Database:
      docker-compose up -d

   2. تشغيل Backend:
      cd backend
      pm2 start src/server.js --name tea-backend

   3. Nginx للـ Frontend
      (اتبع DEPLOYMENT_GUIDE.md)

على Ubuntu Server:
══════════════════

   1. Database:
      cd tea-management-system
      docker-compose up -d

   2. Backend:
      cd backend
      npm install
      npx prisma generate
      npx prisma db push
      pm2 start src/server.js --name tea-backend

   3. Frontend builds جاهزة!
      (من setup-production.bat)

   4. Nginx:
      اتبع DEPLOYMENT_GUIDE.md

   5. SSL:
      sudo certbot --nginx -d teaapp.twaasol.com
